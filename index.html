Technical Architecture & Core Functionalities

VocaPlan is a local-first, offline-capable AI-powered appointment and calendar management application built primarily for Android, with a web-compatible fallback.

Core Functionalities

Calendar Management

Week, Month, and Agenda views with full CRUD capabilities. Complex scheduling, recurring events, multi-day support, and per-instance deletion without breaking chains. Features color-coding, reminders, and "auto-done".

AI Extraction Engine

Processes requests via voice commands, text inputs, audio files, images (like photos of a schedule), and document uploads up to 10 MB.

Voice Processing

Utilizing native Android SpeechRecognizer for standard transcriptions and "agenda command mode" for higher-level commands.

Smart Enhancements

Intelligent titles based on participants and location, AI-fetched national holiday overlays, and geolocation weather widget.

Localization

Dark/light modes with six accent colors, multi-language support, 12h/24h time toggles, and strict IANA timezone awareness.

Technical Architecture

Frontend Layer

UI is built using React 18 and TypeScript, bundled with Vite, and styled with Vanilla CSS. Dates are handled using date-fns.

Mobile Native Bridge

Android app uses Capacitor 6 bridging web frontend with native capabilities. Plugins for Firebase Auth, SQLite-backed local storage, and Android AICore.

Backend & Cloud

Serverless backend built with Node.js and hosted on Google Cloud Run. Leverages Firestore for persistence and Gemini 3.5 Flash for AI processing.

AI Routing Pipeline

Hybrid routing attempts local processing via Android AICore/local NLP first, falling back to cloud-based Gemini 3.5 Flash for heavier tasks like image/document parsing.

Security & Deployment

Firebase Auth persists locally. Validated server-side. Advanced endpoints require premium. Compiled via Gradle, deployed to Firebase EU region.
